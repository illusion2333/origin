# 版本迭代
本章节介绍了长安链版本、产品功能和对应的更新状态时间点。

| 版本                  | 时间点       | 主要内容                   | 
| -------------------- | -------- | -------------------- | 
| 1.0.0                  | 2021.01.27     | 每个区块记录着上一个区块的hash值、本区块中的交易集合、本区块的hash等基础数据。由于每个区块都有上一区块的hash值，区块间由这个值两两串联，形成区块链。                  |
